# Webhook-Automation-Workflow-using-Google-Apps-Script-

Webhook Automation Workflow using Google Apps Script
🚀 Project Overview

This project demonstrates a complete webhook automation workflow using:

Google Apps Script
Webhooks
Hoppscotch API Testing
Google Sheets Automation
JSON Data Handling
Automated Email Response

The workflow receives JSON data through a webhook API, stores it automatically in Google Sheets, and sends an automated email response to the user.

🛠️ Tech Stack
Google Apps Script
Google Sheets
Hoppscotch
Webhook.site
REST API
JSON
⚙️ Features

✅ Webhook API Integration
✅ Real-Time JSON Data Processing
✅ Automatic Google Sheets Data Entry
✅ API Testing using Hoppscotch
✅ Email Automation using GmailApp
✅ Serverless Automation Workflow

📌 Workflow
User sends JSON data using Hoppscotch
Google Apps Script receives webhook request
JSON data is parsed
Data is stored automatically in Google Sheets
Automated email response is triggered
